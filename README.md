# Automated Amazon Operations SLA Report using n8n

This project demonstrates an automated workflow built using n8n to process order data, calculate SLA metrics, and generate automated reports.

## Features

- Extract order data from Excel
- Calculate SLA and operational metrics
- Generate cleaned report
- Convert report to PDF
- Automatically send report via Gmail

## Tools Used

- n8n
- Google Drive
- Gmail
- Excel data processing

## Workflow Steps

1. Scheduled trigger
2. Download Excel file from Google Drive
3. Extract order data
4. Calculate SLA metrics
5. Generate report
6. Send automated email

## Use Case

This workflow helps operations teams automate daily reporting and reduce manual work.
